# Reading List

Curated, ordered by usefulness for *this* project. You don't need all of it now — pull from it per milestone.

## Books
- **Larry Harris — *Trading and Exchanges: Market Microstructure for Practitioners*.** The bible for this project. Chapters on order types, the book, liquidity, and market makers map almost 1:1 onto our milestones. Read the relevant chapter before each milestone.
- **Robert Kissell — *The Science of Algorithmic Trading and Portfolio Management*.** For market impact and execution (Milestone 7).

## Papers / notes
- Maureen O'Hara — *Market Microstructure Theory* (dense, academic; skim for vocabulary).
- Almgren & Chriss — *Optimal Execution of Portfolio Transactions* (the classic on slicing large orders; Milestone 7, stretch).

## Practitioner writing / talks
- **"How to Build a Fast Limit Order Book"** (WK Selph blog) — the canonical intro to the data-structure choices behind Milestone 4/5. Read before designing the book.
- Jane Street / HRT / Optiver tech talks on matching engines and market making (YouTube) — good for vocabulary and interview framing.
- QuantStart, Hummingbot docs — practical market-making mechanics.

## Docs (for the build)
- Python `heapq`, `sortedcontainers` (order-book data structures).
- FastAPI docs (engine API, Milestone 9).
- Next.js + Vercel deploy docs (console, Milestone 9).
- `lightweight-charts` by TradingView (the actual charting lib for the black/red/green console).

## How to use this
For each milestone, the milestone note names *which* of these to read. Log anything genuinely useful as its own note in `10-concepts/` with a `[[backlink]]`.
